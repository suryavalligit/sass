SASS Datatypes (Numbers, Strings, Colors, Lists)

SASS Datatypes - Numbers

Numbers, which may or may not have units, like 12 or 100px.


SASS Datatypes - Strings

Strings, which may or may not have quotes, like "Helvetica Neue" or bold.


SASS Datatypes - Colors

Colors, which can be referred to by their hex representation or by name, like #c6538c or blue, or returned from functions, like rgb(107, 113, 127) or hsl(210, 100%, 20%).


SASS Datatypes - Lists

Lists of values, which may be separated by spaces or commas and which may be enclosed in square brackets or no brackets at all, like 1.5em 1em 0 2em, Helvetica, Arial, sans-serif, or [col1-start].


SASS Datatypes - Boolean

The boolean values true and false.


SASS Datatypes - Null

The singleton null value.


SASS Nested Styles

It is advised to follow BEM(Block Element Module) CSS pattern. To maintain the SASS code in optimized way.

It means that write nested styles by using Class[name].

Class[name] must be in BEM pattern.

Example:
Parent div classname : welcome-card,
Inner title classname : welcome-card_title,
Inner description classname : welcome-card_desc.


SASS Interpolation

What is SASS Interpolation?

SASS Interpolation is a technique to include the result of any SASS Script expression into the stylesheet wherever required using #{} .

To include an expression's result or variable's value in your stylesheet we use the interpolation technique

Interpolation Syntax : #{ }
